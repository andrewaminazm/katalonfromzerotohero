# Katalon Tutorials Blog

A modern, static blog covering **Katalon** tutorials for Web, API, and Mobile test automation — from zero to hero.

## What's inside

- **Getting Started** — Installation, project structure, Spy & Record
- **Web Testing** — Locators, keywords, Object Repository, data-driven tests, assertions, waits
- **API Testing** — REST basics, requests/responses, auth, chaining, data-driven API tests
- **Mobile Testing** — Android & iOS setup, mobile keywords
- **Advanced** — Custom keywords, listeners, CI/CD, reports, BDD (Cucumber), Katalon Platform (TestOps, Runtime Engine)

## Run locally

Open `index.html` in a browser, or serve the folder with any static server:

```bash
# Python
python -m http.server 8080

# Node (npx)
npx serve .
```

Then visit `http://localhost:8080` (or the port shown).
